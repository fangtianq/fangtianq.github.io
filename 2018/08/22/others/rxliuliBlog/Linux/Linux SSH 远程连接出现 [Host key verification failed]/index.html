Linux SSH 远程连接出现 [Host key verification failed]

今早设置了服务器的 SSH 后尝试连接结果出现了这个错误

关键是最后一句话：主机密钥验证失败

想了一下好像是之前也有设置过一次 SSH，所以这次才会不认了呀，删除掉 ~/.ssh/known_hosts 文件就好啦
