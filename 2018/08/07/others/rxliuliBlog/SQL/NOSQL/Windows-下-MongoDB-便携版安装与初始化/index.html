安装服务

每次打开都要手动输入命令指定数据目录并启动 mongod 未免麻烦，我们可以将之添加到系统服务中，以后需要的时候只要启动服务就好了，也能设置自动启动什么的了呢

检查一下系统服务里面是否有 MongoDB，如果没有的话使用管理员权限打开 cmd 再执行一次上面的命令就好了，如果服务没有启动的话就启动它。
