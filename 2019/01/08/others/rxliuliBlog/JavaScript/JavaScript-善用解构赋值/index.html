|| / && 解释

  • || 用来取默认值，避免太多的 if 判断。例如对于 a || b 我们可以认为：如果 a 为空，则赋值为 b
  • && 用来连续执行，避免深层嵌套的 if 判断。例如对于 a || b，我们可以认为：如果 a 不为空，则赋值为 b

注：||/&&` 非常适合简单的默认值赋值，但一旦设置到深层嵌套默认值就不太合适了
